# E-Commerce Marketing Attribution Dashboard

Interactive Power BI dashboard built using real **Google Analytics 4 (GA4)** 
e-commerce data to analyze marketing performance across channels, devices, 
countries, and product categories.

## 📊 Dashboard Features

- **Marketing Channel Revenue Attribution** — Google, Direct, Shop & Others
- **Revenue by Device** — Desktop vs Mobile vs Tablet breakdown
- **Top Revenue Generating Countries** — US, Brazil, Canada, France, India & more
- **Revenue by Product Category** — Treemap visualization (Apparel, Accessories, etc.)
- **Sessions by Marketing Channel** — User engagement per source
- **Interactive Date Filter** — Slice data by custom date range (Nov 2020 – Jan 2021)

## 🛠️ Tools Used

- Power BI Desktop
- Google Analytics 4 (GA4) Dataset
- DAX for measures
- Power Query for data transformation

## 📈 Key Insights

- **Google** is the dominant revenue channel
- **Desktop** accounts for ~58% of total revenue
- **United States** is the top revenue-generating country
- **Apparel** is the highest revenue product category
